## <a href="https://finalytica.co/">finalytica.co</a>

- Building a financial data provider for mutual funds in Sweden and publicly listed companies on NYSE and NASDAQ.
- Offers forecasting of financial statements, DCF's and all-in-all provide a screening tool for stock picking.

<img width="1600" height="1200" alt="fond-data-banner" src="https://github.com/user-attachments/assets/1db67f57-9328-4e8d-a691-3f592421c753" />

<br></br>
- FONDDATA will offer
  - holdings and composition of Swedish mutual funds
  - historical quarterly asset value, liquidity and other assets + liabilities
  - industry, country and instrument type exposure
  - funds with similar portfolio composition (competitors)
  - alternative funds with lower management fees (weighted jaccard)
  - search which funds own a particular set of instruments
  - search across all reported owned instruments (stocks, funds, bonds, derivatives, etc)
  - detect "clone funds"
  - detect most unique funds (anti-similarity)
  - strategy drift over time (convergence/divergence)
  - open rest api
